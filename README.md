# Satellit

Space arcade game written in C

## Getting Started

### Setup
source envsettings

### Shows full help text
satellit

### Prerequisites

Continously tested on Ubuntu 16.04 LTS
Partially tested on Ubuntu 18.04 LTS
Partially tested on Fedora 28
Resolution 1920x1080
