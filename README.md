# Satellit

Space arcade game written in C

## Getting Started

### Setup
```
cd satellit
source envsettings
satellit build
satellit start
```

To show full helptext:
```
satellit
```

### Prerequisites

- Ubuntu 18.04 LTS
- Resolution 1920x1080
- Partially tested on Ubuntu 16.04 LTS
- Partially tested on Fedora 28
